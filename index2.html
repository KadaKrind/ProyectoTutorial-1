Lista de definición

Que es HTML?
Es un Lenguaje de marcado de hipertexto utilizado para crear páginas web.
CSS
Lenguaje de hojas de estilo utilizado para dar estilo y diseño a las páginas web.
JavaScript
Lenguaje de programación utilizado para crear interactividad y dinamismo en las páginas web.
